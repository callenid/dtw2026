DEMO UNIT CSS: px, %, vw, vh, rem (SATU HALAMAN)

👉 Coba resize window browser / putar HP.
👉 Coba ubah di CSS: html { font-size: 16px; } jadi 12px atau 20px.

1) Width: px vs % vs vw

200px (TETAP, tidak ikut layar)
50% (ikut lebar container)
50vw (ikut lebar layar)
px = ukuran mati • % = ikut parent • vw = ikut layar

2) Height: vh (tinggi ikut layar)

Tinggi 30vh (30% tinggi layar)
vh = persentase dari tinggi layar. Cocok buat banner/hero section.

3) Font-size: px vs rem

Teks 24px (tidak ikut root)

Teks 1.5rem (ikut root font-size)

rem = fleksibel & aksesibel (ikut font-size utama di html)

4) Kesimpulan Cepat

  • px → ukuran tetap (icon, border)
  • % → ikut ukuran parent
  • vw → ikut lebar layar
  • vh → ikut tinggi layar
  • rem → ikut font utama (responsive & aksesibel)
